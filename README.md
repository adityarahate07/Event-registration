🎉 Event Registration Form
📌 Project Overview

This project is a simple Event Registration Form created using only HTML. It allows users to enter their personal details, select event options, and submit their registration.

The form is designed using basic HTML elements like:

Forms
Input fields
Fieldsets
Labels
Buttons
🚀 Features
🧑 Personal Information Section
Name
Email
Phone Number
Gender selection
🎟️ Event Details Section
Event selection dropdown
Ticket type (VIP, Standard, Student)
Event date picker
File upload for ID proof
📝 Additional Information
Address textarea
Skills selection (checkboxes)
Reference source dropdown
🔄 Form Controls
Submit button
Reset (Clear) button
🛠️ Technologies Used
HTML5 only (No CSS, No JavaScript)
📂 File Structure
project-folder/
│
├── index.html   # Main HTML file
└── README.md    # Project documentation
▶️ How to Run the Project
Download or copy the HTML file
Save it as index.html
Open the file in any web browser (Chrome, Edge, Firefox)
